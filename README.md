# test-devjs
Rozwiązanie zadania konkursowego Monogo JavaScript Challenge

Wersja Node: v16.31.1 <br>
Wersja NPM: 8.1.2

Uruchomienie skryptu:
<ol>
    <li>W głównym katalogu skryptu używamy komendy w terminalu: <b>npm install</b> do zainstalowania wymaganej paczki.</li>
    <li>Uruchamiamy plik index.js lub wpisujemy w terminalu <b>npm start</b></li>
</ol>

Skrypt korzysta z pakietu <b>node-fetch</b> (plik read-file-from-URI.js). <br>
Możliwe jest także użycie modułu Node do obsługi plików <b>fs</b> co zaprezentowano w pliku read-file.js.
